## 한국디지털미디어고등학교 1학년 1학기 프로그래밍 프로젝트 수행평가(1601 구현민, 1602 권정훈, 1603 김민선)

# 슬기로운 용돈 생활

**슬기로운 용돈 생활**은 매달 받은 용돈과 남은 날짜를 입력하면 현재까지의 지출을 바탕으로 하루 권장 소비액을 계산해 주는 가계부 프로그램입니다.
사용자는 지출 금액과 카테고리를 기록할 수 있으며, 카테고리별 누적 지출과 현재 남은 금액을 확인할 수 있습니다.

## 프로젝트 개요

용돈을 받으면 처음에는 여유롭게 느껴지지만, 식비나 교통비처럼 작은 지출이 반복되면 예상보다 빠르게 돈이 줄어들 수 있습니다.
이 프로그램은 남은 금액과 남은 날짜를 기준으로 하루에 얼마까지 소비하면 좋을지 알려 주어 계획적인 소비 습관을 기르는 데 도움을 주는 것을 목표로 합니다.

## 주요 기능

### 1. 이번 달 용돈 입력

이번 달 받은 용돈과 남은 날짜를 입력하여 예산 정보를 설정합니다.

### 2. 지출 기록

지출 금액과 지출 카테고리를 입력하면 해당 카테고리에 누적 저장됩니다.

| 번호 | 카테고리 |
| -- | ---- |
| 1  | 식비   |
| 2  | 교통비  |
| 3  | 문화생활 |
| 4  | 쇼핑   |
| 5  | 기타   |

### 3. 일일 권장 소비액 확인

현재 남은 금액과 남은 날짜를 기준으로 오늘 사용할 수 있는 권장 소비액을 계산합니다.

```text
일일 권장 소비액 = (이번 달 용돈 - 총 지출 금액) / 남은 날짜
```

### 4. 카테고리별 지출 통계 보기

카테고리별 누적 지출 금액, 총 지출 금액, 현재 남은 금액을 확인할 수 있습니다.

### 5. 프로그램 종료

메뉴에서 5번을 선택하면 프로그램이 종료됩니다.

## 함수 설명

| 함수명                                     | 설명                           |
| --------------------------------------- | ---------------------------- |
| `setBalance(int balance, int remains)`  | 이번 달 용돈과 남은 날짜를 설정합니다.       |
| `setSpending(int amount, int category)` | 지출 금액과 카테고리를 입력받아 지출을 기록합니다. |
| `sumSpending()`                         | 모든 카테고리의 지출 금액을 합산합니다.       |
| `getSpending()`                         | 일일 권장 소비액을 계산하고 출력합니다.       |
| `getStatistics()`                       | 카테고리별 지출 통계와 남은 금액을 출력합니다.   |

## 팀원 및 역할

| 이름  | 역할                      |
| --- | ----------------------- |
| 구현민 | 프로젝트 관리자, 기획자, 설계자, 개발자 |
| 권정훈 | 기획자, 설계자, 개발자, 테스터      |
| 김민선 | 기획자, 설계자, 개발자, 기록자      |

## 기대 효과

이 프로그램을 통해 사용자는 자신의 소비 내역을 카테고리별로 확인하고, 남은 용돈을 기준으로 하루 권장 소비액을 파악할 수 있습니다.
이를 통해 과소비를 예방하고 계획적인 소비 습관을 기를 수 있습니다.
